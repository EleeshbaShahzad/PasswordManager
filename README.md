# Password Vault System (Backend in Python)
## Overview
Password Vault System is a backend-only Python project designed to demonstrate core software engineering and backend development concepts. It focuses on secure user authentication, data handling, and scalable architecture design without using any external libraries or GUI.
The system is built using pure Python with a modular and industry-inspired structure, making it a strong foundation for future expansion into database systems and API-based applications.
## Features
- User Registration system
- Secure Login functionality
- Password hashing using SHA-256
- File-based data storage (JSON)
- Login activity tracking using Queue (DSA concept)
- Fast user lookup using Dictionary (O(1) complexity)
## Concepts Applied
- Object-Oriented Programming (OOP)
- Data Structures (Dictionary, Queue)
- Basic Cryptography (Hashing with SHA-256)
- File Handling (JSON-based storage)
- Modular Backend Architecture
- Clean Code Design Principles
